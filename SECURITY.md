# Security Policy

## Reporting a Vulnerability

Do not disclose vulnerabilities publicly before a fix is available.

Report issues privately to:

- `jesuorobonosakhare873@gmail.com`

Include:

- affected contract/file,
- impact assessment,
- proof of concept,
- recommended remediation.

## Scope

In-scope contracts:

- `src/OrderBookVault.sol`
- `src/LargeCapExecutionHook.sol`
- `src/Executor.sol`

Out-of-scope:

- local demo mocks,
- UI simulation-only logic,
- third-party dependency vulnerabilities not introduced by this project.
